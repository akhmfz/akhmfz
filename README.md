🧭 Tentang Saya

Saya membangun AKHMFZ Analytics — inisiatif open-source untuk mengembangkan indikator, dashboard, dan sistem scoring saham berbasis TradingView Pine Script, dengan fokus khusus pada karakteristik Pasar Modal Indonesia (IDX).

Ketertarikan saya ada di titik temu antara analisis fundamental, desain sistem skoring, dan engineering — bagaimana mengubah data laporan keuangan yang rumit menjadi satu tampilan yang bisa dibaca dalam hitungan detik.


🔭 Sedang mengerjakan: Papan Instrumen — dashboard fundamental & sector-aware scoring untuk saham IDX
🌱 Sedang belajar: metodologi scoring lanjutan (Piotroski F-Score, Altman Z-Score) dan adaptasinya untuk pasar Indonesia
💬 Bisa ditanya soal: analisis fundamental saham, Pine Script v6, desain indikator TradingView
📫 Kontak: buka bagian Connect di bawah
